### **Objective**

This intensive 6-month training program is designed to transform aspiring developers into job-ready **Full Stack Engineers** with expertise in the **MERN stack** and **Generative AI**. The curriculum focuses on building robust, scalable, and secure applications using modern technologies like **TypeScript**, **Docker**, and **cloud platforms**. Through a **project-centric approach**, students will master both frontend and backend development, implement advanced security protocols, and leverage AI tools to accelerate their workflow — culminating in a portfolio of real-world projects ready for employer review.

### **Detailed Curriculum**

#### **1. HTML5**

- Understanding the structure of web pages

- CSS syntax, selectors, and specificity
- Advanced selectors and pseudo-classes (`:nth-child`, `:hover`, `:focus`, etc.)
- Box model, positioning, and float techniques
- Modern layout systems:
  - **Flexbox**: alignment, direction, wrapping, justification
  - **CSS Grid**: grid-template, gap, area, and nested grids
- Responsive web design:
  - Media queries and breakpoints
  - Mobile-first design principles
  - Relative units (%, em, rem, vw, vh)
- Transitions, animations, and transformations
- Using CSS variables and custom properties
- Styling forms, navigation bars, and UI components

#### **3. JavaScript Essentials**

- JavaScript basics:
  - Variables (`let`, `const`), data types, operators
  - Conditional statements and loops
  - Functions (declarations, expressions, arrow functions)
- Arrays and objects manipulation
- DOM manipulation:
  - Selecting elements (`getElementById`, `querySelector`)
  - Modifying attributes, styles, and content
  - Handling events (`click`, `submit`, `keydown`)
- Timing functions (`setTimeout`, `setInterval`)
- Form validation using JavaScript
- Local storage and session storage
- Basic debugging using browser dev tools (console, breakpoints)

#### **4. Version Control (Git & GitHub)**

- Setting up Git and GitHub accounts
- Core Git commands: `init`, `add`, `commit`, `status`, `log`
- Branching and merging workflows
- Handling merge conflicts
- Collaborating on GitHub: forks, pull requests, and issues
- Writing good commit messages and using `.gitignore`
- Publishing and hosting static websites using **GitHub Pages**

#### **5. Generative AI for Developers**

- Introduction to **AI-powered coding assistants**
  - Understanding GitHub Copilot and ChatGPT for coding tasks
- Using Copilot:
  - Autocompletion, comment-driven code generation
  - Refactoring and documentation generation
- Using ChatGPT for:
  - Explaining and debugging code
  - Generating test cases and learning new APIs
- Best practices for prompting AI tools effectively
- Ethical use of AI tools in coding and learning

##### **4. Styling & UI Development**

### **Module 3: Advanced Frontend & State Management**

---

#### **Detailed Topics**

### **Module 6: Advanced APIs & Security Best Practices**

_Duration: Weeks 15–17_

This module teaches you how to secure backend systems through strong authentication, role-based authorization, and safe data handling. You’ll also implement modern OAuth2 flows, handle inter-service communication securely, and follow best practices from the OWASP Top 10.

---

#### **Detailed Topics**

##### **1. Authentication (AuthN)**

- **JWT (JSON Web Token) Fundamentals**
  - Understanding access and refresh tokens.
  - Stateless authentication and token expiration.
- **JWT Implementation**
  - Generating, signing, and verifying tokens using `jsonwebtoken`.
  - Secure token storage (HTTP-only cookies vs local storage).
  - Token rotation and refresh token management.
- **Login/Logout Flow**
  - User registration and credential hashing using `bcrypt`.
  - Secure logout by token invalidation or blacklisting.
- **Advanced Topics**
  - Password reset and email verification flows.
  - Protecting routes using custom Express middleware.
  - Best practices for session expiration and token reuse detection.

---

##### **2. Authorization (AuthZ)**

- **Role-Based Access Control (RBAC)**
  - Defining roles (Admin, Seller, Customer, etc.).
  - Mapping permissions to actions and endpoints.
  - Middleware for role-based route protection.
- **Attribute-Based Access Control (ABAC)**
  - Conceptual introduction to policy-based access rules.
- **Multi-Tenant Security**
  - Isolating user data between tenants (organizations or stores).
  - Tenant-aware JWTs and scoped permissions.
- **API Gateway and Access Layer**
  - Designing secure access layers for internal/external APIs.

---

##### **3. Security & Data Protection**

- **Password Encryption**
  - Using `bcrypt` for password hashing and salting.
  - Validating credentials securely.
- **Input Validation**
  - Using `zod` or `class-validator` to validate request payloads.
  - Preventing injection attacks through strict validation.
- **CORS & CSRF**
  - Configuring CORS properly for frontend-backend communication.
  - Understanding CSRF and mitigation strategies.
- **Common Vulnerabilities (OWASP Top 10)**
  - Overview of major risks: XSS, SQL Injection, CSRF, Insecure Deserialization, etc.
  - Implementing countermeasures in Express apps.
